# Mean & Std. Dev.

#### MEAN

Average value of signal. Add all signals and divide by signal count (N)

$$\LARGE MEAN (\mu) = \frac{1}{N}\sum_{i=0}^{N-1}x_i$$

* N = Sample count
* $$x_i$$ = Sample array

***

#### STANDARD DEVIATION

Uses power deviation (deviation = $$abs(x_i - \mu)$$).  Square all deviations, sum, divide by (N-1) and square root the result.

$$\LARGE \sigma^2 = \frac{1}{N-1}\sum_{i=0}^{N-1}(x_i - \mu)^2$$
