# euler - Sinusoids

### euler

$$e^{j\omega}=cos(\omega) + j sin(\omega)$$

### Sinusoid

$$x(t) = X_m * cos(\omega t + \theta)$$

* $$X_m$$ = peak value or amplitude of signal
* $$\omega$$ = frequency of signal (rad/sec)
* $$\theta$$ = signal phase shift (rad)

#### Complex Sinusoid

$$X= \frac {X_m}{\sqrt{2}}(cos(\theta) + jsin(\theta))$$
