# SNR

#### Signal-to-Noise Ratio (SNR)

<div class="math-left">
$$\LARGE SNR = \frac {signal}{noise} = \frac{MEAN (\mu)}{STD. DEV. (\sigma)}$$
</div>

#### Coefficient-of-Variation (CV)

<div class="math-left">
$$\LARGE CV = \frac {noise}{signal} = \frac{STD.DEV. (\sigma)}{MEAN (\mu)} * 100$$
</div>
